人間の視線の動き

ホームページは人に見られるものであるため、
人にとって見やすいようなものを作る必要があります

そこで、まず人間の視線の動きについて説明してからレイアウトの種類を紹介します。人間の視線の動きには大きく分けて、三つの型が存在します。

  • ◎Z型

    横書きで多くの要素が配置されたものを見ると、人は図のように、Zの形で読み進めていきます。 このような視線の誘導を「Z型」と呼びます。

  • ◎N型

    次は縦書きの場合です。縦書きでは図のようにNの形で読み進めていきます。 このような視線の誘導を「N型」と呼びます。

  • ◎F型

    最後に現在最も使われているものを紹介します。それが図のように「F」の形で読み進めていきます。このような視線の誘導をF型と呼びます。

カラムについて

ホームページのレイアウトの情報を整理して見やすくするために必要な、「カラム」について説明していきます。

情報を整理して見やすくするために、要素と要素の「段組み」を行うことがあります。 実際に雑誌などを見ると写真と文章が見やすい位置にそれぞれ配置されており、このようにホームページでも最適な位置に配置をする必要があります。
この段組みのことを、ホームページ制作においては「カラム」と呼び、カラムがいくつ組み合わさっているかによって、「シングルカラムレイアウト」「マルチカラムレイアウト」というような呼び方をつけています。

シングルカラムレイアウト



左のようにすっきりと縦長に段組みされたレイアウトのことを「シングルカラムレイアウト」といいます。 スマートフォンの普及により、近年は余計な情報は表示しない縦長のシングルカラムレイアウトのサイトが増えてきています。

このレイアウトは特に集客したユーザー向けに、商品などの説明をする「ランディングページ」などは、読むのに集中させることのできる、シングルカラムレイアウトが適しています

マルチカラムレイアウト

マルチカラムレイアウトとはコンテンツの脇にサイドバーを設置して２、３列に分かれたようなレイアウトです。

  • 2カラムレイアウト

  • 3カラムレイアウト

2カラムレイアウトとは、上のように、コンテンツの左、もしくは右にサイドバーを設置したレイアウトのことで, これはコンテンツの左右両方にサイドバーを設置したレイアウトのことです。

これらのマルチカラムレイアウトはスマートフォンでは表しづらいために、主にPCでのWebデザインにおいて用いられます。 レスポンシブデザインで実装する際は、PCではマルチカラムレイアウト、スマートフォンではシングルカラムレイアウトというようにわけて表示することが多いです

スクロール時の動き

ホームページではユーザーのアクションに合わせてレイアウトを変更することができます。
その一例としてスクロール時に特殊な動きを加えるものを紹介します。

ヘッダーを固定するレイアウト

スクロールをした際もヘッダーがそのままの位置で表示されるレイアウトです。 これによって、企業のロゴを常に表示したり、ヘッダーのナビゲーションから、他のページを移動したりするようなページにできます。 

body {
	padding-top: 90px; /* ヘッダーの後ろに要素が隠れないようにするため */
}

header {
	width: 100%; /* 幅いっぱいを指定 */
	height: 50px; /* 高さを50pxに指定 */
	background: #CCC; /* 背景色にグレーを指定 */
	padding: 20px 50px; /* ヘッダーに上下左右それぞれ余白を指定 */
	box-sizing: border-box; /* padding分を含んで幅を100%にするため */
	position: fixed; /* ウィンドウを基準に画面に固定 */
	top: 0; /* 上下の固定位置を上から0pxにする */
	left: 0; /* 左右の固定位置を左から0pxにする */
	display: flex; /* 中の要素を横並びにする */
	align-items: center; /* 中の要素を上下中央に並べる */
}

header .gnav .menu {
	display: flex; /* 中の要素を横並びにする */
}

header .gnav .menu li + li {
	margin-left: 40px; /* メニューそれぞれに間隔をあけるため */
}

main {
	height: 100vw; /* スクロールの演出を見れるようにmainに高さを指定 */
}

これは上のCSSで実装できます。実際にヘッダーを固定するのに重要なのはposition:fixedというCSSです。

このようにユーザーの行うアクションに伴って変化するレイアウトも作ることができます。
