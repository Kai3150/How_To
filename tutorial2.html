基本的なタグ説明

htmlはWebページを作るための言語の一つですが、マークアップ言語であるhtmlは、その名の通りマーク＝『タグ』をつけることで、文章の構造を表現します。ebページを右クリックして「ソースの表示」を選ぶと「<」と「>」で囲まれた部分が『タグ』です。タグの記述方法にはルールがありますが、いくつか覚えれば簡単にWebページが作成できます。

10個の覚えておきたいタグ

1.h1〜h6タグ

見出しを作るときに使用するタグです。h1が一番大きな文字であり、サイトのメインとなる見出しに使われます。h2,h3…h6と数字が大きくなるにつれて小さな文字の見出しとなります。

<h1>メインタイトルなど</h1>
  <h2>サブタイトルなど</h2>
    <h3>さらに小さいタイトル</h3>
      <h4>見出し（h4）<h4>
        <h5>見出し（h5）<h5>
          <h6>見出し（h6）<h6>

2.aタグ

他のページへのリンクを貼ったりする際に使用します。

<a href="遷移先のurlなど">ここの文字をクリックすると遷移します。</a>

3.pタグ

段落を表すためのタグです。タグでタグで囲むと、その文章が一つの段落となって、段落ごとに文章が区切られるための改行を伴います。段落の途中で改行させたい場合はbrタグを使用しましょう。pタグはブッロック要素です。

<p>
  ここには文章を書きます。<br>
  brタグで文章は改行されます。
</p>

4.imgタグ

画像を貼り付ける際に使用するタグです。imgタグはインライン要素です。

<img src="画像url" alt="画像が表示されない時の言葉" width="横幅" height="高さ">

5.tableタグ

テーブル（表）を作成するときに使用するタグです。

<table>

表の開始を示すタグです。中には<tr>・<td>・<th>が入ります。何も入力していない場合では枠線は表示されず、border属性に"1"を指定することで表示されるようになります。

<tr>

table roeの略。テーブルの行に使用します。このタグを使用した分だけ行が追加されます。

<td>

table dateの略でテーブルのセルに使用します。サンプルの中の内容の部分と言えます。

<th>

table headerの略。テーブルの見出しのセルに使用します。

記述例

<table border="1">
    <tr>
        <th>見出しその１</th>
        <th>見出しその２</th>
        <th>見出しその３</th>
    </tr>
    <tr>
        <td>内容</td>
        <td>内容</td>
        <td>内容</td>
    </tr>
    <tr>
        <td>内容</td>
        <td>内容</td>
        <td>内容</td>
    </tr>
</table>

6.brタグ（改行）

文章の途中で改行したいときに使用します。段落内で強制的に改行させる際に使うのですが、連続で使用して空白行を作成するために使わないようにしましょう。


7.ul.ol.liタグ

リストを作成するときに使うタグです。

<ul>

unordered listの略。順番のないリストを作る際に使います。（はじめに付く記号の種類はtype属性で指定することができます。）

<ul type="A">
    <li>項目</li>
    <li>項目</li>
    <li>項目</li>
</ul>
<ol>

ordered listの略。1,2,3〜やa,b,cなど順番のあるリストを作成する際に使います。はじめに付く記号や数字の種類はtype属性で指定することができます。

<ol type="square">
    <li>項目</li>
    <li>項目</li>
    <li>項目</li>
</ol>
<li>

list itemの略。<ol>〇〇</ol>や<ul>〇〇</ul>の中にあたる〇〇の部分に記述します。リストの各項目を作成するときに使います。


8.scriptタグ

文書にJavaScriptなどの「スクリプト」と呼ばれるものを文書内に埋め込んだりするために使うタグです。

この記述がheadタグやbodyタグの中にある場合、JavaScriptを呼び出して何かをしようとしていると認識しましょう。

<script src="script.js">  </script>
