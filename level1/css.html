CSS

ほとんどのWebページではHTMLとCSSという言語を用いて、その見た目が作られていますが、その中でCSSはHTMLのタグで指定した範囲の文字の色や大きさ、背景色や配置などを指定するために用いられる言語です。

  • HTMLだけの場合

  • CSSも加えた場合

上ではHTMLのみで作ったサイトに、CSSを使って背景や文字色を変更したりしています。このようにCSSを使うと、HTMLに装飾を施して、より見やすかったり、見てもらいたいところを強調したりすることができます。

CSSの適用方法

ここでは3つのCSSの適用方法を紹介します

styleタグを用いる方法

styleタグはCSSを記述するためのHTMLのタグです
headタグにいれて、適応するCSSを書いていくのが基本となります。 

            <!DOCTYPE HTML>
            <html>
            <head>
            <title></title>
            <style type="text/css">
            body{
            background-color:red;
            }
            </style>

この方法ではHTMLとCSSが一つのファイルで管理できるため、小規模開発であれば見やすくてよいですね。

style属性を用いる方法

style属性は属性のため他のタグの属性として機能します。 

            <body style="background-color: red;">

この方法では、のちに説明するclassなどをつける必要がないので楽です。 しかし、同じ要素には毎回styleを書かなければならないので、基本的に一部だけの修正などに活躍します。

※属性とは、HTMLの要素に何かしらの設定をつけるもののようなイメージです。
<要素名 属性="属性名">

外部ファイルを参照する方法

headタグの中に他のCSSファイルを参照するようなコードを書いて、CSSはもう一つのファイルに書き込むという方法です。
classなどの属性で指定して、CSSファイルにその内容を記述していきます。 

            <!DOCTYPE HTML>
            <html>
            <head>
            <title></title>
            <link rel="stylesheet" href="ファイル名.css">
            もしくは
            <!DOCTYPE HTML>
            <html>
            <head>
            <style type="text/css">
            @import url("ファイル名.css")

ファイル名.cssに書かれたCSSがHTMLに反映されます。

この方法が最もポピュラーで大規模な開発にも役立ちます。 しかし、HTMLファイルとCSSファイルの二つのファイルが必要となります。

要素の指定

ここでは最もポピュラーな外部ファイルを参照した方法でのCSSの書き方を紹介していきます。

上で説明したような記述をheadタグにして、CSSファイルに内容を書き込んでいきます。
まずは属性により要素の指定を行います。その属性の中でも重要なものにid属性とclass属性があります。

id属性

id属性は要素に識別子を指定します。この属性は1つのドキュメントの中に同じ名前のidを指定することはできません。

class属性

class属性は要素にクラス名を指定します。こちらは1つのドキュメント内に同一のクラスを複数指定することができます。

            <h1 id="midashi">/id指定の場合
            <h1 class="midashi">/class指定の場合

これらを用いて要素を指定して、CSSを反映します。

CSSの記述方法

方法書式記述例
タグで指定タグ名{css}div{font-size:large;}
タグとクラスで指定タグ名.クラス名{css}div.group{background-color:red;}
クラス名で指定.クラス名{css}.group{text-align:center;}
idで指定#id名{css}#group{color:blue;}
複数の要素に適用タグ名1,タグ名2{css}h1,div{text-align:right;}
全ての要素に適用*{css}*{background-color:red;}

上の表のような記述方法があります。

color、text-align等のことをプロパティといい、プロパティとその値で指定された要素を変更します。
